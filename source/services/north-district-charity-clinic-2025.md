### 醫者仁心，福澤社群

健康是每個人最寶貴的財富，但並非每個人都能輕易獲得醫療服務。忠信基金會與藏霞精舍秉持「醫者仁心」的精神，攜手在北區舉辦義診派福活動，為有需要的居民提供免費中西醫診療服務，同時派發愛心福袋，讓關懷與溫暖遍及社區每個角落。

這次活動不僅是一次醫療義診，更是一次社區共融的實踐。透過專業醫護團隊的無私奉獻，以及各界志願者的熱心協助，我們為數百位居民送上了健康諮詢、疾病預防建議，以及實實在在的生活物資，真正做到「義診惠民，派福暖心」。

### 活動亮點

**中西醫聯合義診**
邀請資深中醫師和西醫專科醫生現場義診，為居民提供全面的健康檢查和診療服務。中醫把脈問診，西醫測量血壓血糖，雙管齊下守護居民健康。

**專業健康諮詢**
設立健康諮詢台，由經驗豐富的醫護人員解答居民的健康疑問，提供個性化的保健建議和用藥指導。

**派發愛心福袋**
為參加活動的居民準備了豐富的福袋，包含日常生活用品、保健食品等，讓居民感受到社會的關愛與溫暖。

**慢性病管理指導**
針對高血壓、糖尿病等常見慢性病患者，提供專業的健康管理建議，幫助他們更好地控制病情，提高生活質量。

**中藥湯包贈送**
根據不同體質，贈送適合的中藥保健湯包，讓居民在家也能進行簡單的食療調理。

**量血壓測血糖**
提供免費的血壓、血糖檢測服務，幫助居民及時了解自己的健康狀況，做到早發現早預防。

document.addEventListener('DOMContentLoaded', function() {
    const gallery = document.getElementById('image-gallery');
    const images = Array.from(gallery.querySelectorAll('img'));

    Promise.all(images.map(img => {
        if (img.complete) return Promise.resolve();
        return new Promise(resolve => {
            img.onload = resolve;
            img.onerror = resolve;
        });
    })).then(() => {
        if (window.innerWidth >= 768) {
            groupImagesByHeight();
        }
    });

    let resizeTimer;
    window.addEventListener('resize', function() {
        clearTimeout(resizeTimer);
        resizeTimer = setTimeout(() => {
            if (window.innerWidth >= 768) {
                groupImagesByHeight();
            } else {
                gallery.className = 'mt-8 space-y-6';
                images.forEach(img => {
                    img.style.width = '100%';
                });
            }
        }, 250);
    });

    function groupImagesByHeight() {
        const imageData = images.map(img => ({
            element: img,
            height: img.naturalHeight,
            width: img.naturalWidth
        }));

        const groups = [];
        const tolerance = 0.05;

        imageData.forEach(data => {
            let foundGroup = false;
            for (let group of groups) {
                const refHeight = group[0].height;
                if (Math.abs(data.height - refHeight) / refHeight <= tolerance) {
                    group.push(data);
                    foundGroup = true;
                    break;
                }
            }
            if (!foundGroup) {
                groups.push([data]);
            }
        });

        gallery.innerHTML = '';
        gallery.className = 'mt-8 space-y-6';

        groups.forEach(group => {
            if (group.length === 1) {
                const img = group[0].element.cloneNode(true);
                img.style.width = '100%';
                gallery.appendChild(img);
            } else {
                const row = document.createElement('div');
                row.className = 'flex gap-6';

                group.forEach(data => {
                    const img = data.element.cloneNode(true);
                    img.style.flex = '1';
                    img.style.width = '0';
                    img.style.minWidth = '0';
                    row.appendChild(img);
                });

                gallery.appendChild(row);
            }
        });
    }
});
</script>
